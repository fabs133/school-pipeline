# Example Data

This directory contains **synthetic** school documents for testing and
demonstration. No real student data, teacher materials, or copyrighted
publisher content is included.

All documents use the fictional context "Berufskolleg Beispielstadt".

## Structure

- `tasks/` — Synthetic input documents that mimic real school file dumps
- `output/` — Pipeline-generated artifacts from the example tasks
- `manifests/` — Scanner output showing document classification

## Documents

### DE-BSP (Deutsch)

| File | Role | Description |
|------|------|-------------|
| `IT_B1_A1_Energie.docx` | task | Imperative task with slide requirement |
| `IT_B1_A1_Energie_Texte.docx` | info | Background reading material |
| `IT_B1_A1_Energie.docx.pdf` | duplicate | PDF export of the task |
| `EnergieOneNote.docx` | onenote_export | OneNote container with embed marker |
| `Bewertungsbogen.docx` | info | Grading criteria sheet |
| `Leer.docx` | empty | Empty file |

### WI-BSP (Wirtschaft)

| File | Role | Description |
|------|------|-------------|
| `Aufgaben_Angebot_Nachfrage.docx` | task | Fill-in table exercise |
| `Aufgaben_Angebot_Nachfrage.docx.pdf` | duplicate | PDF export of the task |
| `AngebotNachfrageOneNote.docx` | answer | OneNote export with student notes |

## Try it yourself

```bash
# Scan the example directory
schulpipeline scan examples/tasks/

# Run the pipeline on an example task
schulpipeline run --input examples/tasks/DE-BSP/IT_B1_A1_Energie.docx --yes

# Try different output formats
schulpipeline run --input examples/tasks/DE-BSP/IT_B1_A1_Energie.docx --format docx --yes

# Try different styles
schulpipeline run --input examples/tasks/WI-BSP/Aufgaben_Angebot_Nachfrage.docx --style dark --yes
```

## Regeneration

The documents are generated by `scripts/generate_examples.py`:

```bash
python scripts/generate_examples.py
```
